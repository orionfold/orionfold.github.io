An answer is one thing. The path to it is what tells you whether the next answer will hold.

## The test we locked

Give the model a problem that takes more than one step. A passing model shows those steps in order, so a person can follow the logic and find where it would break, instead of trusting a single number that appeared with no path behind it.

## What happened

Both Kepler and the Patent Strategist show their reasoning step by step. Kepler does it on space-math problems; the Patent Strategist does it on cited legal reasoning. In each case you can read the steps and check that the answer follows from them.

This is the star on the "shows step-by-step reasoning" row for both models. The worked examples live on each model's page.

## The honest part

Shown steps can still be wrong steps. A visible path is not a guarantee of a correct one. What it gives you is the ability to check, to see where a wrong answer went wrong, which a single hidden number never lets you do.

## Rerun it

Give the model a multi-step problem and read the steps it shows. Check that each step follows from the last, and that the final answer matches the path rather than landing on the right number by luck.
