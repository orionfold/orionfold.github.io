People ask how a small studio ships eight AI models. The answer is one desktop and a tight loop.

## One small machine

We run a Spark-class desktop. It is about the size of a lunch box and costs around three thousand dollars. It has enough memory to train and run models that used to need a room full of servers.

## The loop

We start from an open model that others have shared. Then we train it a bit more on one field, like patents or law, so it learns the words and rules of that work. We test it, shrink it so it runs fast, and post it for free.

## Why offline matters

Because the whole loop runs on our own desk, no private text ever leaves the room. For work like legal filings or patient notes, that is the whole point. Your data stays yours.

Eight models are live now, with a test set to score them. All free, all open. More soon.
